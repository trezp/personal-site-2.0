// Personal Projects

Ticktickticket

We were halfway through web developer bootcamp and working on finale projects, twelve of us competing for the attention of one instructor, who would bounce frantically around the room helping whoever yelled the loudest.

It was driving the students insane. "We need an app for this!" they cried.

So I built one. Ticktickticket is a waitlist app built with JavaScript framework MeteorJS. Ticktickticket auto-refreshes in real-time and can be updated immediately by any student from any phone or browser. At the request of the instructor, I added a "waitlist" mode feature that allowed him to display the list on his screen without all the clutter. This application is still used daily in classrooms at PDX Code Guild.

Light Years

Light Years, based on the classic French card game Milles Bornes, was my first foray into Object Oriented JavaScript and gave me the opportunity to learn about different JS design patterns. The earliest iteration of the game was a simple text-based console game that randomly increased and tracked the player's score. Eventually it would take in an array of players, deal all the players cards, and return an array of player objects that had hands, scores, and other methods. The most current protoype of the game was built with Meteor and features a player login where users can view a list of available opponents. The game is currently non-functioning and still a work in progress, but proved invaluable for building up a strong understanding of JavaScript and the principles of programming in general.
