Ticktickticket

Ticktickticket is a waitlist app built with JavaScript framework MeteorJS. Demand was high in my dev bootcamp for one-on-one help with the instructor, leading to long wait times and a lot of confusion. It was driving the students crazy. "We need an app for this!" one of my classmates moaned on a day of longer-than-average wait times.

So I decided to make one. Ticktickticket auto-refreshes in real-time and can be updated immediately by any student from any phone or browser. It continues to be used daily in classrooms at PDX Code Guild.
